<static> snow.util.array.toMap(a, keyName)
From an array of javascript obect, create a map (js object) where the key is the propName value, and the value is the array item. If the propName does not on an item exist, it will ingore the item. 
var myVehicules = [{id:"truck",speed:80},{id:"racecar",speed:200}];
var vehiculeById = snow.util.array.toMap(myVehicules,"id");
// vehiculeById["truck"].speed == 80

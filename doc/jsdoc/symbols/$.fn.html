<static> $.fn.sClick(delegate, options)
This event is a cross touch/mouse click event, that get triggers only if the distance threshold between the down and up.
On touch devices this will also be considered a fast click as it does not bind to the regular touch click event, but to touchstart and touchend In future iterations, we might want make the "touch" case a little bit more advanced (checking touchstart/touchend are from the same element and no drag was initiated)
Defined in: snow.event.js.
Parameters:
{String} delegate
can be the function or a "delegate" selector string if we want this event to be delegated
{Function or Object} options
if function, then, it will be the "callback", but can be an options object
<static> $.fn.sComponent(componentName)
Return the component that this html element belong to. Thi traverse the tree backwards (this html element up to document) to find the closest html element containing the snow.ui component for this name. If a componentName is given then it will try to find the given component. If no componentName is given, then it will return the first component found. For example: 
var myComponent = $(thisDiv).sComponent("myComponent");
Parameters:
{String} componentName
The component name to be match when traversing the tree, if undefined, then, the closestComponent will be return.

<static> $.fn.sDrag(delegate, opts)
Drag event for mouse and touch based user agents. This fires sdragstart, sdrag, and sdragend events on the dragged element, and sdragenter, sdragleave, sdrop on the "overElement" if the opts.draggable is set to true. If opts.draggable is omitted or false, then only the events on the dragged element will be fired. See code sample at /test/test_snow.event.02.sDrag.html for more infor
Defined in: snow.event.js.
Parameters:
{String} delegate
[optional] if present, this will do a delegate (not implemented yet)
{Object} opts
[optional] options and handlers
{Boolean} opts.draggable
[default false] tell if the element is draggable
{String|Function} opts.helper
[default 'original'] if opts.draggable is true, this will determine the strategy for the drag helper. Can be 'clone' 'original' or a function
{Function} opts.start=function(event,dragExtra)
[optional] will be called when the drag is initiated (map to the 'sdragstart' event)
{Function} opts.drag=function(event,dragExtra)
[optional] will be called for every drag event (map to the 'sdrag' event)
{Function} opts.end=function(event,dragExtra)
[optional] called on mouseUp or touch end (map to the 'sdragend' event)
{Number} dragExtra.pageX
the current pageX
{Number} dragExtra.pageY
the current pageY
{Number} dragExtra.startPageX
the pageX from the drag start
{Number} dragExtra.startPageY
the pageY from the drag start
{Number} dragExtra.deltaPageX
the delta between the current pageX and the last one (from the previous drag event)
{Number} dragExtra.deltaPageY
the delta between the current pageY and the last one (from the previous drag event)
{DOMElement} dragExtra.overElement
this is for sdrag and sdragend event, and represent the over element (the drop over element) if the opts.draggable was set to true
{DOMElement} dragExtra.helperElement
this is the element that is used for the drag effect. Could be the same as the draggable element, or a clone or custom element.
